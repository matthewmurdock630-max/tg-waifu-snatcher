# 🌸 Waifu Snatcher Bot

A free, lag-free Telegram bot that serves anime waifus 24/7.

## Deploy (2 min)

1. Fork this repo
2. Sign up at render.com with GitHub
3. New + → Blueprint → pick waifu-bot
4. Set BOT_TOKEN env var (from @BotFather)
5. Done!

Keep alive: free monitor at uptimerobot.com ping the Render URL.
